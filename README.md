# ckassets

Static asset library of maps, images, and tokens for the Candlekeep campaign. Published to GitHub Pages.

## Asset folders

- `maps/` — campaign maps
- `images/` — general images
- `tokens/` — character and creature tokens
- `splash/` — splash artwork

## Viewer

An Express app for browsing and editing assets locally.

```bash
cd viewer
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).
